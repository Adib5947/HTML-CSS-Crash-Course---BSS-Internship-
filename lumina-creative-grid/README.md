# Lumina Creative Grid

An upgraded responsive multi-page portfolio website built using HTML5, CSS3, CSS Grid, Flexbox, and Lightbox.js as part of my HTML & CSS crash course internship training at BSS (Bangladesh Software Solution).

This version is the improved Grid-based version of the original Lumina Creative project.

---

## Features

- Responsive multi-page website
- Modern clean UI design
- CSS Grid gallery layout
- Flexbox navigation and footer layouts
- Responsive media queries
- Interactive image gallery using Lightbox.js
- About page
- Contact form UI
- Reusable utility classes
- Mobile-friendly design
- Accessibility-focused form labels
- Responsive asymmetric portfolio gallery

---

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Responsive Media Queries
- Lightbox2
- Font Awesome
- Google Fonts
