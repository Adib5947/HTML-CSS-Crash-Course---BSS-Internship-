# Lumina Creative

A modern responsive multi-page portfolio website built using HTML5, CSS3, Flexbox, and Lightbox.js as part of my HTML & CSS crash course internship training at BSS (Bangladesh Software Solution).

---

## 🚀 Features

- Responsive multi-page website
- Modern clean UI design
- Flexbox-based layouts
- Responsive media queries
- Interactive image gallery using Lightbox.js
- About page
- Contact form UI
- Reusable utility classes
- Mobile-friendly design
- Accessibility-focused form labels

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Media Queries
- Lightbox2
- Font Awesome
- Google Fonts
