# HTML & CSS Sandbox - Pricing Grid HTML Challenge

This project demonstrates the structure of a **Pricing Grid Layout** using HTML.  
It is part of the **Essential HTML** section from the HTML & CSS learning sandbox.

The project includes multiple pricing cards with plan details, pricing information, feature lists, and call-to-action buttons.

---

## Project Overview

The project includes:

- Pricing card layouts
- Multiple subscription plans
- Plan pricing structures
- Feature lists using unordered lists
- Call-to-action buttons
- Reusable card components

This project helps beginners understand how pricing sections are structured in modern websites.
