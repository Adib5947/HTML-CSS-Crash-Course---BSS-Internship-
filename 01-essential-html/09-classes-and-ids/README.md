# HTML & CSS Sandbox - Classes & IDs

This project demonstrates the usage of **Classes** and **IDs** in HTML for selecting, grouping, and styling webpage elements.  
It is part of the **Essential HTML** section from the HTML & CSS learning sandbox.

---

## Project Overview

The project includes:

- Assigning classes using `class`
- Assigning unique IDs using `id`
- Reusable class names
- Navigation with ID-based links
- Structured webpage sections
- Semantic content organization

This project helps beginners understand how classes and IDs are used for styling, layout design, and webpage navigation.
